## 🚀 Quick Start (Recommended: Hybrid Mode)

1. Copy the entire content of **`MASTER_PROMPT_v3.4.md`**
2. Paste it into a new Grok 4.3 Beta chat
3. Type: **`Activate Grok Imagine Cinematic Studio v3.4`**
4. Choose your workflow:
   - **A** — Full Production Bible (recommended)
   - **B** — Step-by-step control
   - **C** — Quick Scene
   - **D** — Long Sequence Mode (60–120s+)
   - **E** — Custom Agent Selection

## 🧠 Multi-Agent Orchestration Flow

The system uses a shared `studio_state.json`, structured Handoff Packets, and Locked Variables for seamless collaboration between all 15 agents.

**Studio Director** acts as the central orchestrator with full control over agent activation.

## 🏗️ Complete System Architecture

- **Central Hub:** Project Bible with Variable Dependency Mapping
- **15 Enhanced Agents** with v3.4 capabilities
- **Tool Integrations:** `generate_image`, `edit_image`, `search_images`
- **Final Gate:** Quality Assurance Guardian with Comparative Analysis
